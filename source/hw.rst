Hardware Guide
--------------

The SiliconLab Sensor Node is provided with several sensor:

- Si114x: Proximity, ambient light
- MPU-9250: Accelerometer, magnetometer
- AK8963: Magnetometer

The **BGM111** uses I2C interface to communicate with sensors.
